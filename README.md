## 🛡️ SavageStats — What I'm Building Right Now

<div align="center">

  **An offline-first, AI-powered fitness tracker for Android.**

  <a href="https://github.com/Mejji/SavageStats"><img src="https://img.shields.io/badge/🛡️_SavageStats-Check_It_Out-red?style=for-the-badge" alt="SavageStats"/></a>
  <a href="https://github.com/Mejji/SavageStats/releases/latest"><img src="https://img.shields.io/github/v/release/Mejji/SavageStats?label=Latest%20Release&style=for-the-badge&color=red" alt="Latest Release"/></a>

</div>

- 🤖 Runs a **528MB Gemma 3 LLM** entirely on-device via Google MediaPipe — zero cloud, zero subscriptions
- 📸 **Camera food scanner** using a custom TFLite model (2,000 food classes) with USDA macro lookup
- 🗄️ **Offline RAG pipeline** backed by a 411MB USDA nutrition database with FTS4 full-text search
- 🏅 **Savage Rank** progression system — from "Uncooked Noodle" to "Savage God"
- ⚡ Syncs with **Android Health Connect** (Samsung Health, Strava, Fitbit)
- 🔒 **100% private** — all data stays on your device

> *Built with Kotlin, Jetpack Compose, MediaPipe, Room, CameraX, and a lot of sarcasm.*
